# Breast Cancer Prediction
Tree models, namely DecisionTree, RandomForest and XGBoost, were trained on the Wisconsin Breast Cancer dataset to correctly diagnose based on the given features.

## Description
The WBC dataset consists of features computed from a digitized image of a fine needle aspirate (FNA) of a breast mass. These features describe the characteristics of the cell nuclei present in the image.
Dataset is highly dimensional, consisting of numerous features, so I have applied a dimensionality reduction technique to extract key valuable features. I have used tree-based models, namely DecisionTrees,
RandomForest and XGBoost models, tuning the parameters and plotting their learning curve, highlighting the performance of each model on the dataset.
